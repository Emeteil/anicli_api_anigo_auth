Welcome to anicli-api's documentation!
======================================

**anicli-api** библиотека набора парсеров для получения аниме, информации о нём
и видео из различных источников.

Смотрите :doc:`quickstart` чтобы начать работать с ней.

.. note::

   Проект на стадии разработки.

   Большинство парсеров реализованы в обход официальных методов, 100% надёжность не гарантируется.

   Эта библиотека **рассчитана** на применение с **client-side** стороны.

   Если вы хотите с помощью этого решения поднять свой сервис, то используйте anilibria/animevost REST api или обратитесь к хостерам таких видео.
